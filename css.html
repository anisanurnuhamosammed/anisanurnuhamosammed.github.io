Ode ai CSS

Con il color: #ff6666; dipingi emozioni,

con il font-size crei dimensioni.

Il font-family racconta lo stile,

e con il font style tutto è più sottile.

Infine il font-weight da peso e spessore,

ai testi che vibrano nel nostro cuore.

/* File: styles.css */ body { font-family: Arial, sans-serif; margin: 0; padding: 20px; background-color: #ffffff; color: #333; } .poem { text-align: center; margin: auto; max-width: 600px; } h1 { color: #2aa198; font-size: 2.5em; margin-bottom: 20px; } p { font-size: 1.2em; line-height: 1.6; color: #555; } .highlight { color: #ff8c00; font-style: italic; background: linear-gradient(to right, #ff8c00, #ffcc00); -webkit-background-clip: text; -webkit-text-fill-color: transparent; } strong { font-weight: bold; font-style: italic; }
